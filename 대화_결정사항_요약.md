# 대화·결정사항 요약
- 기존 통합판에서 실제 자동연구 시작을 눌러도 변화가 없어 보이는 문제를 수정한다.
- 연구 시작 즉시 진행 상태를 표시한다.
- 후보 엔진을 사용자가 직접 선택해 추천 엔진으로 바꿀 수 있게 한다.
- 선택적으로 자동 엔진 교체도 지원한다.
- 자동교체는 현재 챔피언보다 검증 종합점수가 좋아야 한다.
- 백서와 대화 결정사항을 파일에 계속 포함한다.


- v27: 자동연구 1% 멈춤 대응을 위해 별도 프로세스 구조로 변경. CMD 창은 실행 시 숨김.


- v29: 검은 CMD 창 없이 실행하도록 2_실행.vbs로 변경. 실행 실패 원인은 startup.log와 error.log에 자동 기록.


- v30: 사용자의 선택에 따라 GitHub Actions Windows 빌드 방식으로 전환. GUI EXE와 연구 엔진 EXE를 함께 배포하며 CMD 창을 숨김.

- v31: 사용자가 요청한 '엔진 자동교체 후 계속 자동연구' 기능 추가.
- 반복 횟수, 대기시간, 중지 기능을 제공하며 매 주기 결과를 저장.
- 현재는 3개 고정 엔진 재검증 방식이며 신규 변이 엔진 자동생성은 별도 고급 단계로 구분.
