太炅 Lotto Lab v30 GitHub 업로드 및 Windows EXE 빌드

[중요]
압축을 푼 뒤 안쪽 파일과 폴더를 저장소의 가장 위쪽(루트)에 업로드하세요.
.github 폴더도 반드시 함께 업로드해야 합니다.

1. GitHub 저장소 접속
   hieroking / Taegyeong-Lotto-Lab

2. Code 화면에서 Add file → Upload files

3. 이 패키지의 파일과 폴더를 모두 업로드
   app.py
   research_worker.py
   requirements.txt
   .github 폴더
   기타 문서

4. Commit changes 누르기

5. 상단 Actions 선택

6. 왼쪽에서 Windows EXE Build v30 선택

7. Run workflow → 초록색 Run workflow

8. 완료되면 실행 결과 화면 아래 Artifacts에서
   Taegyeong-Lotto-Lab-v30-Windows 다운로드

9. 다운로드 압축을 풀고
   Taegyeong_Lotto_Lab_v30.exe 실행

주의:
Taegyeong_Lotto_Lab_v30.exe와 research_worker.exe는 반드시 같은 폴더에 있어야 자동연구가 동작합니다.
