# 太炅 Lotto Lab v25.2 백서

추천과 실제 워크포워드 자동연구를 통합한다.

핵심 변경:
1. 연구 시작 무반응처럼 보이던 고비용 조합 선택 루프를 고속화한다.
2. 시작 즉시 상태·진행률을 표시한다.
3. 검증 결과를 엔진관리 화면에 전달한다.
4. 수동 챔피언 선택과 보호형 자동교체를 모두 지원한다.
5. 자동교체는 현재 챔피언보다 종합점수가 높은 경우에만 적용한다.
6. 실제 검증되지 않은 성과 수치를 표시하지 않는다.


## v27 변경
- QThread 자동연구를 QProcess 별도 프로세스 방식으로 교체
- 진행률 JSON 통신 및 120초 감시 타이머 추가
- 실행 시 CMD 창 숨김
- 실행 파일은 1_설치.bat, 2_실행.bat 두 개로 제한


## v29 변경
- 실행기를 VBS 무콘솔 방식으로 교체
- startup.py 진단 래퍼 추가
- startup.log 및 error.log 자동 생성
- 시작 실패 시 Windows 오류창 표시


## v30 GitHub Windows EXE 구조
- GitHub Actions windows-latest에서 PyInstaller 빌드
- GUI EXE는 windowed 모드로 콘솔 미표시
- 자동연구 엔진은 별도 research_worker.exe로 빌드

## v31 연속 자동연구
- 자동연구 완료 후 다음 연구를 자동으로 시작
- 최대 반복 횟수 1~10,000회
- 반복 간 대기시간 0~3,600초
- 중지 버튼과 현재 주기 표시
- 매 주기 결과를 AI 엔진관리로 전달
- 자동교체 ON이면 더 좋은 엔진으로 갱신 후 다음 연구 진행
- UTF-8 프로세스 환경 고정으로 한글 로그 깨짐 완화
